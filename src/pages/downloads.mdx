jCanvas requires [jQuery](https://jquery.org/) version 1.9 or newer. If you're
already using jQuery, you can just add jCanvas to your page (via `<script>`) or import via ESM/CJS/AMD.

jCanvas is licensed under the MIT license, which you can view on
[GitHub](https://github.com/caleb531/jcanvas/blob/master/LICENSE.txt).

## Supported Browsers

- Chrome 80+
- Edge 80+
- Firefox 74+
- Safari 13.4+
